# 💰 Expense Tracker (MERN Stack)

A modern full-stack Expense Tracker web application built using the MERN stack. The application enables users to securely manage their daily expenses, visualize spending patterns, and organize financial records through an intuitive dashboard.

---

## 🚀 Features

- 🔐 User Authentication (JWT)
- 🔒 Password Encryption using bcrypt
- ➕ Add Expenses
- ✏️ Edit Expenses
- 🗑️ Delete Expenses
- 📋 View Expense History
- 🔍 Search Expenses
- 🏷️ Filter by Category
- 📅 Date-wise Expense Tracking
- 📊 Category-wise Expense Visualization (Pie Chart)
- 🌙 Dark/Light Theme
- 🚪 Secure Logout
- 📱 Responsive User Interface
